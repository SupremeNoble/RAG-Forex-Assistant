# 🧠 RAG Forex Assistant

A Retrieval-Augmented Generation (RAG) chatbot that provides forex insights using uploaded documents, technical indicator guides, and market data.

## 🧩 Tech Stack
- Python 3.10
- LangChain
- ChromaDB
- Gradio
- Hugging Face Llama 3 Inference API (for responses)

## 🏗️ Project Structure
```
rag_forex_assistant/
├── gradio_app.py
├── ingest.py
├── retriever.py
├── llm.py
├── requirements.txt
└── .gitignore
```
